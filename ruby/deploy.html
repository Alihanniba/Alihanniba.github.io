Ruby on rails ubuntu14.04部署方案

1. 更新系统的存储库数据库和安装的软件包
apt-get update
apt-get upgrade
2. 安装Ruby依赖
sudo apt-get install git-core curl zlib1g-dev build-essential libssl-dev libreadline-dev libyaml-dev libsqlite3-dev sqlite3 libxml2-dev libxslt1-dev libcurl4-openssl-dev python-software-properties libffi-dev
3. 安装RVM
sudo apt-get curl
curl -L https://get.rvm.io | bash -s stable

or

\curl -L https://get.rvm.io | bash -s stable --ruby
4. 安装RVM的环境依赖
rvm requirements
5. 安装Ruby
rvm install ruby

如果想在Ubuntu上安装多个Ruby版本，使用rvm作为默认的Ruby版本管理

rvm use ruby --default

or

rvm --default use ruby-2.3.1
6. 检查当前成功安装的Ruby版本
ruby -v
7. 升级ruby gem管理器
gem update --system
8. 显示gemsets列表
rvm gemset list

show

gemsets for ruby-2.3.0 (found in /usr/local/rvm/gems/ruby-2.3.0)
   (default)
=> global
9. 看看全局gemset中安装的gem列表
rvm gemset use global
gem list
10. 获取已过时的宝石列表
gem outdated

更新它们

gem update
11. 禁用文档步骤来加快gem安装
echo "gem: --no-document" >> ~/.gemrc
12. 拉取项目
git clone https://github.com/alihanniba/rails-on-awesome.git
13. Install Bundler
gem install bundler
14. Install rails
gem install rails
rails -v

or get newest beta version

gem install rails --pre
rails -v

or get a specific version, for example

gem install rails --version=5.0.0
rails -v
15. 测试项目是否正常运行
cd /rails-on-awesome
bundler
gem server
16. 接下来就是服务器配置及反向代理配置
