    关于我--编码界的一名小学生

    花一个无所 11/26/2016 生活笔记

    题图: pixabay

    # 思考了几分钟

    这个自我介绍的开场还是不知道该怎么写.想起现在在做的事儿,总是有人说我是个大奇葩.以前,我是想当舞蹈家的,混着混着就莫名其妙投身编码界了.

    # 初三那年

    迷上了街舞,总是想象着长大了出去跳舞多么拉风,那时候打听到上高中了有舞蹈课,会有老师教,我天真的相信了.于是上高中了才知道,童话里都是骗人的,学校老师教的是民族舞、古典舞、芭蕾舞.那时候幻想着,都是舞,都一样,能挨着边也挺好,于是顺带着学了2年古典舞,也不能叫专攻,学校管这叫艺术生,耐不住舞蹈老师好看呀,于是不管不顾的把舞蹈作为主修,声乐辅修.

    # 后来

    各种原因,终于没能跳成,上大学进了旅游管理专业,反正我记得我是没报这个专业,军训完了跟班上小伙子混熟了也就这样了.四年时间说长不长说短不短.到大三,进了北京一家酒店实习,本来6个月的实习期,过了3个月的某一天,我开始思考人生,思考啊~思考啊~思考到最后得出了一个结论----反正我特么就是不想干了,哈哈哈~~~.也许是我顿悟了,于是第二天直接去酒店辞职了.

    # 后来

    在宿舍呆了几天,没工作也不行啊,因为喜欢斯诺克,于是去一家台球俱乐部面试了,想着还能学下球,完了人家问我小台能不能清台,那个我就激动了,心里想着"我多屌啊,清台这玩意儿",嘴上说着要不要打一局试试,结果人家没接我的话----这就尴尬了!

    # 后来

    直接去上班了,晚班,凌晨2点下班,店里都没人了,然后和小伙伴打到早上8点,那还是说傅家俊等下9点要来宣传,不能被抓了现场,离开后就再也没去那家俱乐部了,也不亏,打一场球50一小时呢.

    # 再后来

    就是回深圳了,买了一张北京到深圳的火车票,走的那天去了一趟王府井图书馆,"书中自有黄金屋",去瞧瞧有啥发家致富的路子,起先准备搞设计、建筑,因为看到了一本CAD的书,感觉挺牛比的,也不知道啥玩意儿,打电话给学校学艺设的小学妹一问,才知道我挂了电话依然不知道这是啥玩意儿,说是得电脑配置好,不然制图很忧伤,于是我就放弃了,因为我电脑写个PPT都卡得飞起,制个蛋的图.后来转了好一会才发现我职场的启蒙书,封面上的图打动了我,于是花了一百块买了职场上的第一本书,咋个现在只要79了,我明明记得当初给了收银的100块,就给我找了一块,呐,就是下面这本书

    # 然而

    仅仅在当晚的火车上看了一眼就再也没有下文了,火车离开北京那一刻,难免有些心酸,10个同学一起来实习,俺一个人灰溜溜的走了---两年吧,两年以后,劳资会回来的,记住这天--2014年8月20号.

    # 转眼快到毕业

    该写论文,找工作了,才想起来那本书,这才拿出来啃,啃了一段时间,从网上扒了一个妙味课堂的demo就去面试了,把其中的js都删掉免得被问起来我不会而尴尬,理所当然的处处碰壁,最后一个公司问:"你会写PHP吗?","PHP是什么?",然后被轰走了.不过后来还是电话说可以过去实习,没有薪水,没有补贴,不过依然很高兴,面包总是会有的.

    # 刚好那天

    高中的小伙伴看到我在学这个,特意过来学校说要教我java,我当然不亦乐乎,语言什么的无所谓,能coding就行,于是学了半个月的java基础就出去面试了,对的,我又出去面试了!就是这么不怕死,那时候完全不懂web service、jdbc、mysql,当然结果很乐观,起码跟面试官聊得挺开心的.她还跟我说加油呢!

    # 2015年6月18号,21岁,毕业了,没辙了。

    毕业前夕广撒网,万一碰见没长眼的(伯乐)呢,广投长江上中下游一带及以南包括但不限于杭州、成都、重庆、长沙、广州、深圳等地,最终荣获两个面试邀请,一个广州一个深圳,于是打包了大学四年的东西准备南下了.

    衣服被子可以丢,球杆不能忘.

    # 后来

    背着书包,提着电脑,拖着箱子一个人去广州面试,起初CTO是不打算要我的,因为基础实在太差,后来CEO一看小伙儿长得还挺好,勉勉强强收下了.后来就在CTO(大哥)的带领下茁壮成长了(大哥,现在依然还是大哥,大哥不要打我),接下来就跟着公司以及公司的小伙伴(老蔡(默哀...)、小达达、沛文、西瓜、美人、伟伦、一杰、四哥)一起从广州搬到深圳南山,再搬到深圳车公庙.

    一年时间,写前端、PHP的时间一半一半,多亏了技术部小伙伴的无私奉献、慷慨帮助以及无数个漫漫长夜的陪伴(集体宿舍),一起加班,一起debug到天明.

    # 终究

    我还是来了北京!

    跟着马哥有肉吃, 还得带小宾宾、骚浪贱去做大人............
    不管怎样,就是来了,该咋样咋样吧!

    # 给自己定个小目标，三十岁之前年薪百万吧。O(∩_∩)O哈哈哈~
