    滴滴一下

    花一个无所 9/17/2017 生活笔记

    题图：Streetwill

    # 周二去滴滴入职

    在滴滴大厦外面磨蹭到9点15才进去，入职仪式完了领电脑跟临时工卡，完了差不多12点，项目组的人来消息说等着吃饭，于是又辗转到湾流跟项目组会和。临时分配了个工位，然后吃饭。完了后，开始熟悉公司制度，邮箱，内网账号等等，6点的时候拿到了项目权限，开始熟悉代码，分配到的第一个项目是做一个内部控制系统。吃完晚饭，开始熟悉产品原型，分析需求文档。9点多的时候，跟leader知会一声，撤了。从未想过在滴滴的第一天是这样子度过。

    以前公司入职的时候，总会打几天酱油，然后有个同事带着一起做项目。现在长大了，进的公司也长大的。一切都需要自己去把控，leader只负责传递任务需求，而我们只需要完成就好，过程不重要，重要的是结果。谁先做出来，谁就是大哥。

    # 周四上午拍员工卡

    尴尬的我不会摆姿势，被摄影师调教了好久。完了想去领显示器，被告知我的流程才走了一半，需要邮件通知才能领。而跟我一天入职的同事，申请还在我之后，邮件却下来了，正好可以顺道取回去，不禁感叹，就是这么制度，就是这么有尿性。

    # 晚上的时候

    leader来问项目进度，可怜我才写了两天啊，问不加班的情况下国庆之前项目能上线不，比我早进来一个礼拜跟我一同写这个项目的同事立马摇摇头说不可能，这场面就有点尴尬了。确实都是新接触这个项目，后台跟产品都是远程，需求不明确，文档不明确，时间确实不好把控。奈何，老大就是要国庆前看到产品，压力山大。

    # 上周一周都是差不多晚上9点多10点离开公司，周一原公司离职，周二滴滴入职。

    可能是散漫惯了，前公司基本没怎么加过班，最晚也就是8点离开公司，突然转变，还真有点不习惯。想想毕业那一年，基本上都是晚上十一二点下班，那时候打了鸡血的状态。不知道现在还能不能回得去，还真有点期待，毕竟又是同一群更优秀的人共事。

    # “不要仅成为一个专家”。

    这句话是我第一个领导教导我的。教我怎么看待工作与生活。

    # “写注释，写注释。说了多少遍写注释，莫装逼”。

    这句话是我第二个领导常跟我讲的。教我怎么写代码。

    如今，该是自己出去了。
